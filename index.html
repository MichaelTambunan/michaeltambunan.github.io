tentang saya

Saya lahir di Balige pada tahun 2001. Sejak kecil, saya sudah memiliki ketertarikan terhadap dunia teknologi dan ingin tahu bagaimana segala sesuatu bekerja. Pada usia 18 tahun, saya mulai mempelajari coding dan merancang website sederhana untuk keluarga dan teman-teman. Setelah lulus dari sekolah menengah atas, saya semakin giat belajar programming melalui internet, kelas online, dan bergabung dengan komunitas-komunitas yang ada di Facebook dan Twitter. Selama ini, saya telah mengembangkan kemampuan dalam beberapa bahasa pemrograman seperti HTML, CSS, JavaScript, dan PHP. Saya sangat senang mencari tahu tentang teknologi terbaru dan mempelajari hal-hal baru yang dapat meningkatkan kemampuan saya dalam dunia programming. dan terus mengembangkan kemampuan saya dalam bidang ini. Jika Anda tertarik untuk bekerja sama dengan saya atau memiliki pertanyaan lain, jangan ragu untuk menghubungi saya melalui email atau media sosial yang tercantum di halaman kontak.
